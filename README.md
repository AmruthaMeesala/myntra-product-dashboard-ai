🛍️ Myntra Product Analysis Dashboard (AI-Assisted)

📌 Overview
This project explores building a data dashboard using Claude AI to analyze Myntra product data.

The goal was to understand how AI can help in structuring data, generating visualizations, and designing dashboards.
---
⚙️ Tools Used
- Microsoft Excel  
- Claude AI  
---
 📊 Key Features
- Product category analysis  
- Pricing and discount insights  
- Rating analysis  
- Stock availability tracking  
---
🚀 Approach
This dashboard was built entirely using AI assistance.

AI was used to:
- Structure the dataset  
- Suggest visualizations  
- Design the dashboard layout  
---
💡 Key Takeaway
AI can generate dashboards quickly, but understanding the data is essential to extract meaningful insights.
